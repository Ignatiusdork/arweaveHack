8pin is a permaweb app and tutorial to showcase the basic functionality of an Arweave development platform. Using common web development toolkits and graphql, you can build completely decentralized applications that are deployed on the permaweb forever.

Interested in building an Arweave dapp, checkout this tutorial. And then register for the build program and get started. Arweave.build
